Wat is dit?



NEABBS is een Nederlandse bulletin board uit de jaren tachtig, na veertig jaar stilte heropend. Geen website, geen app: je belt in zoals toen — alleen gaat dat nu over SSH.



Binnen vind je prikborden, een bestandenhoek, een babbelbox en de andere bellers. Je SSH-sleutel is je identiteit; bij de eerste keer inbellen kies je een naam. Het is een spel én een hangplek — hoe diep je graaft, bepaal je zelf.


Inbellen (Mac, Linux, Windows 10+):


ssh neabbs.com
